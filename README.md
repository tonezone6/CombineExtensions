# CombineExtensions

A collection of Swift Combine operators:

### Handle events:
`onSubscribe(perform:)`, `onReceive(perform:)`, `onComplete(perform:)`

### Map custom failure:
`tryMap(failure:)`

### Catch error:
`catch(to:onWeak:replaceWith:)`

### Assign result:
`assign(to:onWeak:)`

### Converts:
`convertToResult`
