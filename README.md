# CombineExtensions

A collection of Combine operators:

### Handle events:
`onReceive(_ handler:)`, `onComplete(_ handler:)`

### Catch:
`catch(assignTo:on:replaceWith:)`

### Assign both result and error:
`assign(to:and:on:)`

### Custom error types:
`tryMap(failure:)`

### Converts:
`convertToResult`
